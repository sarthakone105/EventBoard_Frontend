EventBoard Frontend (Vanilla)
===============================

Simple vanilla frontend for EventBoard (HTML + Bootstrap + Chart.js).

How to use
---------
1. Download or clone the project.
2. Open any `*.html` file in your browser (index.html is the dashboard).
3. The frontend talks to the backend at https://eventboard-backend.onrender.com. If you host the backend elsewhere, update BASE in js/main.js.

Features
--------
- Dashboard (summary, leaderboard, event averages)
- Players page (list, create, delete)
- Events page (list, create)
- Scores page (list, create)
- Dark / Light toggle (saved to localStorage)
